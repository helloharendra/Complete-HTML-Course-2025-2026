Complete List of HTML5 Tags with Explanations

This document explains every HTML5 tag in simple words so even beginners can understand easily.

Basic Structure

<!DOCTYPE> → Defines the document type. In HTML5 it declares the page as HTML5.

<html> → Root element of an HTML page. All content is inside this.

<head> → Contains information about the page (metadata, title, links, styles).

<title> → Sets the page title shown in the browser tab.

<body> → The main content of the page visible to users.

Metadata

<base> → Sets a base URL for all relative links.

<link> → Connects external files like CSS or icons.

<meta> → Provides extra info like description, keywords, or character set.

<style> → Defines internal CSS styling rules.

Sectioning

<header> → Defines the top section of a page or article, often with logos and navigation.

<footer> → Bottom section, usually with copyright, contact info, or links.

<nav> → Represents navigation links like menus.

<article> → Self-contained content, e.g., blog posts.

<section> → Groups content thematically.

<aside> → Content related to main content, like a sidebar.

<address> → Contact information like address, email, phone.

<main> → The main unique content of the page.

<h1> to <h6> → Headings from largest (h1) to smallest (h6).

<hgroup> → Groups headings together (rarely used).

Grouping Content

<div> → Generic container for grouping content.

<p> → Paragraph of text.

<hr> → Horizontal line separating content.

<pre> → Preformatted text preserving spaces and line breaks.

<blockquote> → A block of quoted text.

<ol> → Ordered list (numbers).

<ul> → Unordered list (bullets).

<li> → List item inside ul/ol.

<dl> → Description list (terms and definitions).

<dt> → Term in a description list.

<dd> → Definition of a term in a description list.

<figure> → Groups media content like images with captions.

<figcaption> → Caption for a <figure>.

Text-Level Semantics

<a> → Creates a hyperlink to another page or location.

<abbr> → Defines an abbreviation, shows full form on hover.

<b> → Bold text (without importance).

<bdi> → Isolates text direction for mixed languages.

<bdo> → Overrides text direction (e.g., left-to-right).

<br> → Line break inside text.

<cite> → Reference to a creative work (like book, movie).

<code> → Displays programming code snippets.

<data> → Associates content with machine-readable value.

<dfn> → Marks the defining instance of a term.

<em> → Emphasized text (usually italic).

<i> → Italic text (used for alternate voice).

<kbd> → Represents keyboard input.

<mark> → Highlights text with yellow background.

<q> → Short inline quotation.

<rp> → Fallback text for browsers not supporting ruby annotations.

<rt> → Explanation or pronunciation in ruby text.

<ruby> → Small text annotations above characters (used in Asian languages).

<s> → Strikethrough text (no longer correct).

<samp> → Sample output from programs.

<small> → Smaller, less important text.

<span> → Inline container for styling.

<strong> → Important text (bold, with meaning).

<sub> → Subscript text.

<sup> → Superscript text.

<time> → Represents date/time values.

<u> → Underlined text (stylistic).

<var> → Variable in math or programming.

<wbr> → Suggests where line breaks may occur.

Edits

<del> → Text that has been deleted (strikethrough).

<ins> → Text that has been inserted (underlined).

Embedded Content

<area> → Defines clickable area inside an image map.

<audio> → Embeds audio files.

<img> → Embeds images.

<map> → Defines image map for clickable areas.

<track> → Provides subtitles/captions for video/audio.

<video> → Embeds video files.

SVG and MathML

<svg> → Defines Scalable Vector Graphics.

<math> → Displays mathematical notations.

Table Content

<table> → Creates a table.

<caption> → Title or explanation for a table.

<colgroup> → Groups table columns for styling.

<col> → Defines a single column’s properties.

<tbody> → Groups table body rows.

<thead> → Groups table header rows.

<tfoot> → Groups table footer rows.

<tr> → Table row.

<td> → Table data cell.

<th> → Table header cell.

Forms

<form> → Form for user input.

<fieldset> → Groups form elements.

<legend> → Caption for a fieldset.

<label> → Label for form input.

<input> → Single-line input field.

<button> → Clickable button.

<select> → Drop-down list.

<datalist> → Provides autocomplete options for input.

<optgroup> → Groups options inside a select.

<option> → Option inside a select or datalist.

<textarea> → Multi-line input field.

<output> → Displays calculation results or outcomes.

<progress> → Progress bar indicator.

<meter> → Gauge for measurement values.

Interactive Elements

<details> → Expandable content box.

<summary> → Heading for a details box.

<dialog> → Popup dialog box.

Scripting

<script> → Embeds JavaScript code.

<noscript> → Content shown when JavaScript is disabled.

<canvas> → Used for drawing graphics with JavaScript.

Deprecated/Obsolete Tags (Legacy HTML, not recommended)

<acronym> → Old way to define abbreviations (use <abbr>).

<applet> → Old way to embed Java applets (no longer supported).

<big> → Enlarges text (use CSS instead).

<center> → Centers text (use CSS instead).

<dir> → Directory list (use <ul>).

<font> → Font styling (use CSS).

<frame> → Part of frameset, outdated method of layouts.

<frameset> → Defines a set of frames (obsolete).

<noframes> → Alternate content for no-frame browsers.

<strike> → Strikethrough (use <s> or CSS).

<tt> → Teletype text (use CSS).

End of Document: You now have explanations for all ~145 HTML5 tags (including obsolete ones).
