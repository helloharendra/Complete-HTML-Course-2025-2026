## Basic Structure Tags

These tags form the foundation of every HTML document.

| Tag | Description | Example |
|-----|-------------|---------|
| `<!DOCTYPE>` | Defines the document type and version of HTML | `<!DOCTYPE html>` |
| `<html>` | The root element that contains all other HTML elements | `<html lang="en">...</html>` |
| `<head>` | Contains meta information about the document | `<head><title>My Page</title></head>` |
| `<title>` | Sets the title of the web page (shown in browser tab) | `<title>My Website</title>` |
| `<body>` | Contains all the visible content of the web page | `<body>All visible content goes here</body>` |

## Metadata Tags

These tags provide information about the HTML document itself.

| Tag | Description | Example |
|-----|-------------|---------|
| `<base>` | Specifies a base URL for all relative URLs | `<base href="https://example.com/">` |
| `<link>` | Links external resources like CSS files | `<link rel="stylesheet" href="styles.css">` |
| `<meta>` | Provides metadata like character set or viewport settings | `<meta charset="UTF-8">` |
| `<style>` | Contains CSS code for styling the document | `<style>body { color: red; }</style>` |

## Sectioning Tags

These tags help organize content into logical sections.

| Tag | Description | Example |
|-----|-------------|---------|
| `<address>` | Provides contact information | `<address>Contact us at example@email.com</address>` |
| `<article>` | Represents a self-contained composition | `<article>A news story goes here</article>` |
| `<aside>` | Contains tangentially related content | `<aside>This is a sidebar with related info</aside>` |
| `<footer>` | Defines a footer for a document or section | `<footer>Copyright 2023</footer>` |
| `<header>` | Represents introductory content | `<header><h1>Website Title</h1></header>` |
| `<h1>` to `<h6>` | Defines HTML headings (h1 is most important) | `<h1>Main Heading</h1>` |
| `<hgroup>` | Groups a set of heading elements | `<hgroup><h1>Main</h1><h2>Subtitle</h2></hgroup>` |
| `<main>` | Specifies the main content of a document | `<main>The primary content of the page</main>` |
| `<nav>` | Defines a section of navigation links | `<nav><a href="/home">Home</a></nav>` |
| `<section>` | Defines a section in a document | `<section>A chapter of content</section>` |

## Grouping Content Tags

These tags help group related content together.

| Tag | Description | Example |
|-----|-------------|---------|
| `<div>` | A generic container for grouping content | `<div>A division or section</div>` |
| `<p>` | Defines a paragraph of text | `<p>This is a paragraph of text.</p>` |
| `<hr>` | Represents a thematic break (horizontal line) | `<hr>` |
| `<pre>` | Defines preformatted text (preserves spaces) | `<pre>Text with   spaces preserved</pre>` |
| `<blockquote>` | Indicates an extended quotation | `<blockquote>A long quotation from another source</blockquote>` |
| `<ol>` | Defines an ordered (numbered) list | `<ol><li>First item</li></ol>` |
| `<ul>` | Defines an unordered (bulleted) list | `<ul><li>List item</li></ul>` |
| `<li>` | Defines a list item | `<li>An item in a list</li>` |
| `<dl>` | Defines a description list | `<dl><dt>Term</dt><dd>Description</dd></dl>` |
| `<dt>` | Defines a term in a description list | `<dt>HTML</dt>` |
| `<dd>` | Provides the description of a term | `<dd>HyperText Markup Language</dd>` |
| `<figure>` | Represents self-contained content | `<figure><img src="image.jpg"></figure>` |
| `<figcaption>` | Defines a caption for a figure | `<figcaption>Caption for the figure</figcaption>` |

## Text-level Semantics Tags

These tags define the meaning or appearance of text.

| Tag | Description | Example |
|-----|-------------|---------|
| `<a>` | Defines a hyperlink | `<a href="https://example.com">Click here</a>` |
| `<abbr>` | Defines an abbreviation or acronym | `<abbr title="World Health Organization">WHO</abbr>` |
| `<b>` | Makes text bold without extra importance | `<b>This text is bold</b>` |
| `<bdi>` | Isolates text with different directionality | `<bdi>Text with different directionality</bdi>` |
| `<bdo>` | Overrides the current text direction | `<bdo dir="rtl">This text is right-to-left</bdo>` |
| `<br>` | Produces a line break | `Line one<br>Line two` |
| `<cite>` | Defines the title of a creative work | `<cite>The Great Gatsby</cite> by F. Scott Fitzgerald` |
| `<code>` | Displays computer code | `<code>console.log('Hello');</code>` |
| `<data>` | Links content with machine-readable translation | `<data value="123">Product Name</data>` |
| `<dfn>` | Represents the defining instance of a term | `<dfn>HTML</dfn> is the standard markup language` |
| `<em>` | Marks text with stress emphasis (typically italic) | `I <em>really</em> mean it` |
| `<i>` | Represents text in an alternate voice (typically italic) | `<i>This is italic text</i>` |
| `<kbd>` | Represents keyboard input | `Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy` |
| `<mark>` | Represents highlighted text | `This is <mark>important text</mark> to notice` |
| `<q>` | Defines a short inline quotation | `She said, <q>Hello there!</q>` |
| `<rp>` | Provides fall-back for ruby annotations | `<ruby>漢 <rp>(</rp><rt>kan</rt><rp>)</rp></ruby>` |
| `<rt>` | Defines pronunciation in ruby annotations | `<ruby>漢 <rt>kan</rt></ruby>` |
| `<ruby>` | Represents pronunciation annotations | `<ruby>漢 <rt>kan</rt></ruby>` |
| `<s>` | Renders text with strikethrough | `<s>This text is no longer correct</s>` |
| `<samp>` | Represents sample program output | `<samp>File not found.</samp>` |
| `<small>` | Renders small print (like copyright) | `<small>Copyright 2023</small>` |
| `<span>` | A generic inline container | `<span style="color:red">Red text</span>` |
| `<strong>` | Indicates strong importance (typically bold) | `<strong>Warning!</strong> This is important` |
| `<sub>` | Defines subscript text | `H<sub>2</sub>O` |
| `<sup>` | Defines superscript text | `E = mc<sup>2</sup>` |
| `<time>` | Represents a specific period in time | `<time datetime="2023-12-25">Christmas</time>` |
| `<u>` | Renders underlined text | `<u>This text is underlined</u>` |
| `<var>` | Represents a variable | `The variable <var>x</var> represents the unknown value` |
| `<wbr>` | Represents a word break opportunity | `VeryLongWord<wbr>BreakOpportunity` |

## Edits Tags

These tags represent edits to the document.

| Tag | Description | Example |
|-----|-------------|---------|
| `<del>` | Represents deleted text (strikethrough) | `<del>This text has been deleted</del>` |
| `<ins>` | Represents inserted text (underline) | `<ins>This text has been inserted</ins>` |

## Embedded Content Tags

These tags embed external content into the document.

| Tag | Description | Example |
|-----|-------------|---------|
| `<area>` | Defines a clickable area inside an image map | `<area shape="rect" coords="0,0,50,50" href="link.html">` |
| `<audio>` | Embeds sound content | `<audio controls><source src="audio.mp3"></audio>` |
| `<img>` | Embeds an image | `<img src="image.jpg" alt="Description">` |
| `<map>` | Defines an image map with clickable areas | `<map name="workmap"><area shape="rect" coords="0,0,50,50"></map>` |
| `<track>` | Specifies text tracks for media elements | `<track src="subtitles.vtt" kind="subtitles" srclang="en">` |
| `<video>` | Embeds a video player | `<video controls><source src="movie.mp4"></video>` |

## SVG and MathML Tags

These tags are for vector graphics and mathematical notation.

| Tag | Description | Example |
|-----|-------------|---------|
| `<svg>` | Container for SVG (Scalable Vector Graphics) | `<svg width="100" height="100"><circle cx="50" cy="50" r="40"></svg>` |
| `<math>` | Container for MathML content | `<math><mi>x</mi><mo>+</mo><mi>y</mi></math>` |

## Table Content Tags

These tags are used to create and structure tables.

| Tag | Description | Example |
|-----|-------------|---------|
| `<table>` | Defines a table | `<table><tr><td>Cell</td></tr></table>` |
| `<caption>` | Defines a table caption | `<caption>Monthly Sales</caption>` |
| `<colgroup>` | Specifies a group of columns for formatting | `<colgroup><col span="2" style="background-color:red"></colgroup>` |
| `<col>` | Specifies column properties | `<col style="width:50%">` |
| `<tbody>` | Groups the body content in a table | `<tbody><tr><td>Data</td></tr></tbody>` |
| `<thead>` | Groups the header content in a table | `<thead><tr><th>Header</th></tr></thead>` |
| `<tfoot>` | Groups the footer content in a table | `<tfoot><tr><td>Total</td></tr></tfoot>` |
| `<tr>` | Defines a row in a table | `<tr><td>Cell 1</td><td>Cell 2</td></tr>` |
| `<td>` | Defines a standard data cell | `<td>Table cell data</td>` |
| `<th>` | Defines a header cell | `<th>Column Header</th>` |

## Forms Tags

These tags are used to create interactive forms.

| Tag | Description | Example |
|-----|-------------|---------|
| `<form>` | Creates an HTML form for user input | `<form action="/submit" method="post">Form elements</form>` |
| `<fieldset>` | Groups related elements in a form | `<fieldset><legend>Personal Info</legend>...</fieldset>` |
| `<legend>` | Defines a caption for a fieldset | `<legend>Personal Information</legend>` |
| `<label>` | Defines a label for an input element | `<label for="name">Name:</label>` |
| `<input>` | Creates an input field | `<input type="text" name="username">` |
| `<button>` | Creates a clickable button | `<button type="submit">Submit</button>` |
| `<select>` | Creates a drop-down list | `<select><option value="1">Option 1</option></select>` |
| `<datalist>` | Specifies pre-defined options for an input | `<datalist id="browsers"><option value="Chrome"></datalist>` |
| `<optgroup>` | Groups related options in a drop-down list | `<optgroup label="Group Name"><option>Option</option></optgroup>` |
| `<option>` | Defines an option in a drop-down list | `<option value="value">Display Text</option>` |
| `<textarea>` | Defines a multi-line text input | `<textarea rows="4" cols="50">Default text</textarea>` |
| `<output>` | Represents the result of a calculation | `<output name="result">0</output>` |
| `<progress>` | Represents completion progress of a task | `<progress value="70" max="100">70%</progress>` |
| `<meter>` | Represents a scalar measurement | `<meter value="0.6">60%</meter>` |

## Interactive Elements Tags

These tags create interactive UI components.

| Tag | Description | Example |
|-----|-------------|---------|
| `<details>` | Creates a disclosure widget | `<details><summary>More Info</summary>Content</details>` |
| `<summary>` | Specifies a visible heading for details | `<summary>Click to expand</summary>` |
| `<dialog>` | Defines a dialog box or window | `<dialog open>This is a dialog box</dialog>` |

## Scripting Tags

These tags are used for adding scripts to documents.

| Tag | Description | Example |
|-----|-------------|---------|
| `<script>` | Embeds or references executable code | `<script>alert('Hello!');</script>` |
| `<noscript>` | Defines content if scripts are not supported | `<noscript>Your browser doesn't support JavaScript</noscript>` |
| `<canvas>` | Used to draw graphics via scripting | `<canvas id="myCanvas" width="200" height="100"></canvas>` |

## Deprecated Tags

These tags are outdated and should not be used in new projects.

| Tag | Description | Alternative |
|-----|-------------|-------------|
| `<acronym>` | Was used for acronyms | Use `<abbr>` instead |
| `<applet>` | Was used to embed Java applets | Use `<embed>` or `<object>` instead |
| `<big>` | Was used to make text bigger | Use CSS instead |
| `<center>` | Was used to center content | Use CSS instead |
| `<dir>` | Was used for directory lists | Use `<ul>` instead |
| `<font>` | Was used for font styling | Use CSS instead |
| `<frame>` | Was used to define a frame within a frameset | No modern alternative |
| `<frameset>` | Was used to define a set of frames | No modern alternative |
| `<noframes>` | Was used for browsers that don't support frames | No modern alternative |
| `<strike>` | Was used for strikethrough text | Use `<del>` or CSS instead |
| `<tt>` | Was used for teletype text | Use `<code>` or CSS instead |
